# Data credits

Hooyi uses movie metadata and images supplied by [The Movie Database (TMDB)](https://www.themoviedb.org).

> This product uses the TMDB API but is not endorsed or certified by TMDB.

Streaming availability data is supplied through TMDB's watch-provider endpoint and requires attribution to [JustWatch](https://www.justwatch.com).

TMDB and JustWatch names, logos, data, and images remain the property of their respective owners. Hooyi uses them only to identify the data sources in this non-commercial portfolio project.
